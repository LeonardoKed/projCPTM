Este repositório faz parte do desenvolvimento do Sistema de Cadastro de Efluentes, criado em uma parceria acadêmica entre a FATEC e a Companhia Paulista de Trens Metropolitanos (CPTM).

O projeto tem como objetivo apoiar a CPTM no controle, registro e acompanhamento de efluentes gerados em suas operações, contribuindo para processos mais eficientes, seguros e alinhados às normas ambientais.


Estrutura do Repositório:

CPTM_app_frontend  
Interface do sistema, onde ocorre o cadastro, visualização e interação do usuário.

CPTM_app_backend  
API responsável pelo processamento, armazenamento e validação dos dados de efluentes.
